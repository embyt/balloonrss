Channel Settings Dialog

The channel settings dialog allows the user to configure the RSS channels from which the messages are retrieved. The main channel settings dialog offers the following actions:

  • New: This allows you to add a new RSS channel and will open the channel edit dialog box.
  • Delete: The currently selected channel will be deleted.
  • Edit: This opens the channel edit dialog box to modify the currently selected channel.
  • OK / Cancel: Pressing the OK button will exit the dialog and change all your settings. The Cancel button will close the dialog and discard all changes.

If there are pre-configured multi-user channels in your environment they will be indicated with a locked symbol . They can only be modified by the system administrator as described in advanced usage.

Edit RSS Channels Dialog

The edit RSS channel settings dialog allows the user to configure the settings of the selected or new RSS channel. The settings have the following meaning:

  • Link: The URL of the RSS channel to retrieve.
  • Priority: The pre-defined priority of the channel. The effective priority that is used by the program will be calculated based on this value and the user's click ratio.
  • Mark RSS messages of this channel as read at application startup: If this checkbox is selected BalloonRSS will only display those RSS messages that arrive newly after program start. Therefore, all messages that are already present at the time of the first channel retrieval at application startup will be marked as read and will not be displayed then.

BalloonRSS also supports RSS channels that require a user name and password via HTTP authentication. Please see at advanced usage for more details on this topic.
