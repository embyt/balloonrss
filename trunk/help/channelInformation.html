Channel Information Dialog

The channel information dialog displays data associated to the configured RSS channels in table format. For each channel, the following information is displayed:

  • Title: The URL of the RSS channel.
  • Set Priority: The pre-defined priority associated to the channel by the user configuration.
  • Click Rate: The percentage of RSS messages that were clicked by the user to display the detailed information within the web browser.
  • Effective Priority: The effective priority, which is calculated by BalloonRSS based on the pre-defined priority and the click rate. The calculation depends on the influence balance between the 2 input characteristics that is determined by the application settings.
  • Unread: The number of RSS messages that is queued and ready to be displayed.
  • Total: The number of RSS messages that is currently present online.
  • Last Update: The timestamp of the last successful retrieval of the RSS channel.
